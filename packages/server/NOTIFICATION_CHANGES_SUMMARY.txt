                              COMMENT TESTER
================================================================================

ÉTAPE 1: Redémarrer le serveur
  $ # Arrête et redémarre le serveur

ÉTAPE 2: Obtenir un token
  $ curl -X POST http://localhost:3000/api/auth/login \
      -H "Content-Type: application/json" \
      -d '{"email":"client@example.com","password":"password123"}'
  
  Copie le "accessToken"

ÉTAPE 3: Ouvrir l'exemple
  - Ouvre packages/server/NOTIFICATION_DUAL_CLIENT_EXAMPLE.html

ÉTAPE 4: Tester (4 clics)
  1. Colle le token
  2. Clique "Connecter Client 1" → "✅ Connecté"
  3. Clique "Connecter Client 2" → "✅ Connecté"
  4. Remplis le formulaire et clique "Envoyer Notification"

RÉSULTAT ATTENDU:
  ✅ Notification apparaît immédiatement dans Client 1
  ✅ Pas besoin de refresh!
  ✅ Pas d'erreurs "invalid signature"
